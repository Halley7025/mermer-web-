## 📖 游戏背景

在遗忘之地的深处，有一座名为"梅墨域"的神秘地下城。传说中，这里曾是古代文明的试炼场，封存着无数宝藏与危险。

你是一位失去记忆的冒险者，被困在这个扭曲的领域中。每一次死亡，你都会回到起点，但记忆中的碎片却在逐渐拼凑。唯一的出路，就是不断挑战自己，突破十层地牢的考验。

**记忆币系统**：每次冒险都会留下"记忆币"，这些是你意识的碎片。收集记忆币可以解锁更强大的能力，帮助你在下次轮回中走得更远。

## ✨ 游戏特色

### 🎯 核心玩法
- **8大职业系统**：从近战战士到远程法师，每个职业都有独特的战斗风格
- **职业进阶**：击败 Boss 后可以选择职业进阶，解锁更强大的技能
- **Roguelike机制**：每次死亡重新开始，但记忆币可以累积
- **无尽模式**：通关后挑战无限层数，考验你的极限

### 🎨 视觉风格
- **纯像素绘制**：每个角色、敌人都是手工像素艺术
- **8种敌人类型**：从史莱姆到恶魔，每种敌人都有独特外观
- **动态战斗特效**：粒子系统、伤害飘字、技能特效

### 🎵 音乐系统
- **Web Audio API** 实时音频合成
- **双轨背景音乐**：普通关卡与 Boss 战有不同的音乐氛围
- **动态音效**：根据游戏状态自动切换

### 🎁 丰富的奖励系统
**23种奖励道具**，包括：
- **攻击强化**：增加伤害、攻速、暴击
- **生存能力**：增加生命、护甲、生命偷取
- **特殊效果**：金币磁铁、嗜血狂暴、多重射击、复活

## 🎯 游戏指南

### 基础操作
- **WASD / 方向键**：移动角色
- **鼠标点击**：攻击敌人
- **空格键**：冲刺（快速位移）
- **ESC**：暂停 / 继续游戏

### 游戏流程
1. **选择职业**：根据你的游戏风格选择初始职业
2. **战斗房间**：消灭所有敌人进入下一层
3. **奖励房间**：每层战斗后选择一个奖励道具
4. **Boss关卡**：第 3、6、10 层是 Boss 关卡
5. **职业进阶**：击败 Boss 后可以选择进阶路线
6. **通关/无尽**：第 10 层通关后可进入无尽模式

### 游戏技巧
- **走位优先**：利用冲刺躲避敌人和子弹
- **利用边界**：敌人会从四周刷新，保持中心位置
- **奖励搭配**：优先选择与你职业匹配的奖励
- **Boss战术**：Boss 体型巨大但有固定攻击模式
- **无尽挑战**：Boss 血量和体型会随层数递增

## 🛠️ 技术栈

- **纯原生开发**：HTML5 + CSS3 + JavaScript (ES6+)
- **Canvas 2D 渲染**：960x640 分辨率，60 FPS
- **Web Audio API**：实时音频合成
- **无依赖**：零外部库，纯浏览器原生 API
- **响应式设计**：适配不同屏幕尺寸

直接访问：[https://halley7025.github.io/mermer-web-/](https://halley7025.github.io/mermer-web-/)

```bash
# 克隆仓库
git clone https://github.com/Halley7025/mermer-web-.git

# 进入游戏目录
cd mermer-web-/web

```
