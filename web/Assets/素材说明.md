# Assets 素材文件说明

## 📁 文件分类

### 🎮 角色相关
- **spritesheet_characters.png** - 角色精灵表（主要使用）
  - 包含多种角色职业
  - 适合本游戏的战士、法师、游侠
  
- **roguelikeChar_magenta.png** - Roguelike角色（品红背景）
  - 单个角色精灵表
  - 可用于不同职业的变体
  
- **roguelikeChar_transparent.png** - Roguelike角色（透明背景）
  - 单个角色精灵表
  - 推荐使用（透明背景更好集成）

### 🗺️ 地图/背景
- **tilemap.png** - 地图瓦片
  - 地牢地面、墙壁、装饰
  - 可用于绘制战斗房间背景
  
- **tilemap_packed.png** - 地图瓦片（打包版）
  - 优化过的地图瓦片
  
- **spritesheet_tiles.png** - 瓦片精灵表
  - 各种地牢元素
  - 门、宝箱、楼梯等

### 🛡️ 道具/物品
- **genericItems_spritesheet_colored.png** - 彩色道具精灵表
  - 武器、盔甲、药水、金币等
  - 可用于奖励系统
  
- **genericItems_spritesheet_white.png** - 白色道具精灵表
  - 单色版本，可自行着色

### 🎯 UI图标
- **iconsDefault.png** - 默认图标集
  - UI按钮、状态图标
  - 技能图标
  
- **iconsDouble.png** - 双倍尺寸图标
  - 更大的图标版本
  - 适合高分辨率显示

### 📋 精灵表集合
- **sheet_black1x.png** - 黑色主题精灵表（1x）
  - 完整游戏素材集合
  - 包含角色、敌人、道具、地图
  
- **sheet_black2x.png** - 黑色主题精灵表（2x）
  - 2倍大小版本
  - 更清晰的显示效果
  
- **sheet_white1x.png** - 白色主题精灵表（1x）
  - 完整游戏素材集合（白色边框）
  
- **sheet_white2x.png** - 白色主题精灵表（2x）
  - 2倍大小版本

## 🎨 推荐使用方案

### 方案一：使用角色精灵表（推荐）
```javascript
// 使用 spritesheet_characters.png
// 适合多职业系统
```

**优点**：
- ✅ 多种角色可选
- ✅ 风格统一
- ✅ 尺寸适中（16x16）

**适合职业**：
- 战士：第一行第一个角色
- 法师：第二行第一个角色  
- 游侠：第三行第一个角色

### 方案二：使用完整精灵表
```javascript
// 使用 sheet_white2x.png 或 sheet_black2x.png
// 包含所有素材
```

**优点**：
- ✅ 包含角色、怪物、道具、地图
- ✅ 风格完全统一
- ✅ 一站式解决方案

**缺点**：
- ⚠️ 需要精确裁剪坐标
- ⚠️ 文件较大

### 方案三：混合使用
```javascript
// 角色：spritesheet_characters.png
// 怪物：从sheet中提取史莱姆、骷髅等
// 道具：genericItems_spritesheet_colored.png
```

**优点**：
- ✅ 灵活性最高
- ✅ 可选择最佳素材

## 📐 精灵尺寸

根据文件名推测：
- 角色：16x16像素
- 道具：16x16像素
- 地图瓦片：16x16像素
- 大型精灵表：需要根据实际尺寸计算

## 🎯 集成建议

1. **快速测试**：先用 `roguelikeChar_transparent.png` 测试单个角色
2. **完整方案**：使用 `spritesheet_characters.png` 实现职业系统
3. **优化版本**：最后切换到 `sheet_white2x.png` 获得最佳效果

## 🔍 怪物素材

从精灵表中可以找到：
- **史莱姆** - 适合普通敌人
- **骷髅** - 适合近战敌人  
- **幽灵** - 适合远程敌人
- **Boss** - 使用大型怪物精灵（如龙、恶魔等）
